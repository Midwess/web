title: Configuration
description: CLI flags and environment variables for PgPaw.
---

PgPaw uses CLI flags with environment variable fallbacks.

## Server flags

| Flag | Env | Default | Purpose |
| --- | --- | --- | --- |
| `--host` | `CACHE_HOST` | `127.0.0.1` | HTTP bind host. |
| `--port` | `CACHE_PORT` | `8080` | HTTP bind port. |
| `--data-dir` | `CACHE_DATA_DIR` | `./cache-data` | Replica data directory. |
| `--max-connections` | `CACHE_MAX_CONNECTIONS` | `8` | pglite multi-process pool max. |
| `--cache-size-bytes` | `CACHE_SIZE_BYTES` | `268435456` | Query result cache budget. |
| `--cors-origin` | `CORS_ORIGIN` | unset | Origin, comma list, or `*`. |

## Upstream flags

| Flag | Env | Default |
| --- | --- | --- |
| `--pg-host` | `UPSTREAM_HOST` | `127.0.0.1` |
| `--pg-port` | `UPSTREAM_PORT` | `5432` |
| `--pg-user` | `UPSTREAM_USER` | `postgres` |
| `--pg-password` | `UPSTREAM_PASSWORD` | empty |
| `--pg-database` | `UPSTREAM_DATABASE` | `postgres` |
| `--publication` | `UPSTREAM_PUBLICATION` | `cache_server_pub` |
| `--slot` | `UPSTREAM_SLOT` | `cache_server_slot` |
| `--sslmode` | `UPSTREAM_SSLMODE` | `disable` |

`--sslmode` accepts `disable`, `prefer`, `require`, and `verify-full`.

## JWT flags

| Flag | Env | Default |
| --- | --- | --- |
| `--jwt-secret` | `JWT_SECRET` | unset |
| `--jwt-public-key` | `JWT_PUBLIC_KEY` | unset |
| `--jwt-jwks-url` | `JWT_JWKS_URL` | unset, not implemented |
| `--jwt-role-claim` | `JWT_ROLE_CLAIM` | `role` |

Set only one JWT key source.
