---
title: API
description: PgPaw HTTP endpoints for snapshots, live streams, and health.
---

| Method | Path | Purpose |
| --- | --- | --- |
| `POST` | `/query` | Accept a read-only SQL query. Public queries redirect to a snapshot. Private queries return inline JSON. |
| `POST` | `/query?live=true` | Stream initial snapshot plus row deltas. |
| `GET` | `/q/{hash}/{version}` | Fetch cached public snapshot. |
| `GET` | `/healthz` | Return replica status and watermark. |

## Request body

```json
{
  "sql": "select id, email from users where id = 7"
}
```

## Error envelope

Errors return JSON from `CacheError::envelope()`. Status mapping in code:

| Condition | Status |
| --- | --- |
| Rejected SQL or parse error | `400` |
| Replica halted | `503` |
| Invalid/missing auth | `401` |
| RLS/database denial | `403` |
| Other server error | `500` |

## Health

```bash
curl http://127.0.0.1:8080/healthz
```

Healthy response:

```json
{ "status": "ok", "watermark": 12345678 }
```

Halted response:

```json
{ "status": "halted", "reason": "..." }
```
