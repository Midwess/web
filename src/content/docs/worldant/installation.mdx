---
title: Installation
description: Install the Worldant runtime package and optional supervisor CLI.
---

Install the package that runs inside your Node app:

```bash
npm install @midwess/worldant
```

Add it to app startup:

```ts
import { install } from "@midwess/worldant";

await install();
```

`install()` calls `createWorld()`, registers the result with `@workflow/core`, starts the world, and returns the world handle.

## Supervisor CLI

Install the CLI when the app should run under Worldant supervision:

```bash
npm install -g @midwess/worldant-cli
```

Then start your app through the supervisor:

```bash
worldant run -- npm start
```

Use `start` for single-process foreground supervision. Use `run` for daemon-backed app management.

## Platform notes

The cloned source ships native packages for Node apps on macOS and Linux. The supervisor code uses Unix sockets and Unix process primitives. Deno and Windows are not the current target.

## Verify setup

```bash
node -e "import('@midwess/worldant').then(m => console.log(typeof m.install))"
worldant ps
```

`worldant ps` starts or contacts the daemon and prints active supervised apps when any exist.
