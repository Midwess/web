---
title: Installation
description: Install the PgPaw binary through Cargo or npm.
---

Install from crates.io:

```bash
cargo install pgpaw
```

Or install the npm package that provides prebuilt binaries for supported platforms:

```bash
npm install -g pgpaw
```

The CLI binary in source is named `cache-server` through Clap metadata, but package usage is `pgpaw`.

## Requirements

- A reachable upstream PostgreSQL database.
- Superuser or equivalent privileges for one-time `pgpaw init` when changing `wal_level`, creating the publication, and installing the DDL event trigger.
- A local data directory for the embedded pglite-rs replica.

## Verify

```bash
pgpaw --help
pgpaw serve --help
pgpaw init --help
```
