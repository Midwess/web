---
title: Reading runs
description: Planned workflow run status and stream surfaces.
---

Workflow runs are read by run id.

```text
GET /__worldant/v1/runs/:runId
GET /__worldant/v1/runs/:runId/stream
```

The status endpoint returns the durable run state. The stream endpoint returns
encoded chunks for workflows or calls that produce stream output.

Runs are durable because their progress is stored in PgPaw:

```text
workflow_runs
workflow_events
workflow_steps
workflow_waits
jobs
stream_chunks
```

The active process is disposable. The database state is not.
