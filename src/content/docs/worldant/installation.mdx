---
title: Installation
description: Current installation status for the active Worldant rebuild.
---

Worldant is actively in progress. There is no stable installation path yet.

The intended local shape is:

```bash
worldant build
worldant run
```

`worldant build` validates the world directory and writes `.dist` snapshots.
`worldant run` starts the daemon and serves the latest valid snapshots,
reactive streams, MCP/HTTP surfaces, and npm remote-function protocol.

Until the runtime is implemented, treat this repository as a design and
development workspace.
