---
title: GET /q/{hash}/{version}
description: Fetch a cached public query snapshot.
---

Snapshot URLs are only produced by public `POST /query` calls.

```bash
curl http://127.0.0.1:8080/q/{hash}/{version}
```

## Hit

```http
HTTP/1.1 200 OK
Content-Type: application/json
ETag: {hash}:{version}
Cache-Control: public, max-age=259200
```

Body is the JSON array returned by the replica query.

## Miss

```http
HTTP/1.1 404 Not Found
Content-Type: application/json
```

```json
{ "name": "NotFound", "message": "unknown cursor" }
```

## Cache semantics

The code sets `max-age=259200`, which is 72 hours. The URL includes the query fingerprint and version. When relevant upstream data changes, PgPaw issues a new URL for the same SQL.
