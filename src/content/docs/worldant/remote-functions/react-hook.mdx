---
title: Client adapters
description: Ragent and other clients consume Worldant through the generated package.
---

Worldant does not define frontend state patterns.

Ragent owns generated UI and can import the generated package or call
`worldant-client` directly. Other adapters can do the same.

```text
Worldant owns:
  contract generation
  package endpoint
  remote function transport
  PgPaw truth
  reactive subscriptions

Ragent/client owns:
  UI source
  rendering
  local component state
  user interaction ergonomics
```

The shared boundary is committed database truth. A client action calls a
mutation, PgPaw commits the change, and subscriptions update every authorized
consumer.
