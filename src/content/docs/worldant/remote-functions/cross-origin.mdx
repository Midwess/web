---
title: Cross origin
description: Consume Worldant from another deployment.
---

Cross-origin mode is for clients that install and call a world hosted somewhere
else.

```ini
@world:registry=https://world.example.com/__worldant/v1/npm
```

```bash
pnpm install @world/chess
```

```ts
import { configure, chessBoard } from "@world/chess";

configure({ token: sessionToken });

for await (const board of chessBoard({ gameId })) {
  console.log(board);
}
```

The generated package includes default connection metadata, but callers may
override `baseUrl`, `prefix`, and `token`.
