---
title: Remote functions
description: Worldant's planned npm protocol for generated typed clients.
---

Worldant exposes the active world as an installable package.

The package contains generated wrappers for:

- commands,
- mutations,
- workflows,
- reactive subscriptions,
- streaming calls.

Generated packages depend only on `worldant-client`. They do not depend on the
daemon internals, PgPaw, workflow kernel, source files, or Ragent.

```ts
import { chessBoard, move, previewMove } from "@world/chess";

const preview = await previewMove({ gameId, from: "e2", to: "e4" });
await move({ gameId, from: "e2", to: "e4" }, { key: "move:e2:e4" });

for await (const board of chessBoard({ gameId })) {
  console.log(board);
}
```

Ragent can consume this package. So can a browser app, CLI, service, or external
agent.
